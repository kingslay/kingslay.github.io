Obc 类变量的实现方式

Dec 25th, 2013

obc中是没有类变量的。只能在实现类里增加一个静态变量。然后，调用initialize方法，来初始化。这个静态变量只能能在实现类里用，所以别的类也要用的话，那就要在头文件里增加一个类方法，用于返回这个类变量。(如果是在头文件增加静态变量的话，那就全局变量了。)

static LoginDelegate *loginDelegate;

+(void)initialize
{
  loginDelegate = [[LoginDelegate alloc] init];
}

IOS代码cookie

Nov 20th, 2013

一、想点击UIViewController的任意地方，就可以dismiss keyboard有这几种方法

  1. 在VC中override以下方法:

     -(void)touchesBegan:(NSSet *)touches withEvent:(UIEvent *)event
 {
     [self.view endEditing:YES];
 }

    如果获取view比较困难的时候，可以换成调用

     -(void)touchesBegan:(NSSet *)touches withEvent:(UIEvent *)event
 {
     [[UIApplication sharedApplication] sendAction:@selector(resignFirstResponder) to:nil from:nil forEvent:nil]; 
 }

    或

     -(void)touchesBegan:(NSSet *)touches withEvent:(UIEvent *)event
 {
     [[[UIApplication sharedApplication] keyWindow] endEditing:YES]
 }
